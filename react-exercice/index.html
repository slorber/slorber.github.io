Exercice instructions

The purpose of this exercice is to reproduce the exact behavior of the application on the right. To get started, we will give you a CodePen with the proper environment setup.

Note that finishing this exercice is not a requirement! Do the most you can with the time you can invest in it and your current level.

The point of this exercice is to test your capacity to learn new technologies (if they are new to you), and your sense of observation and details.

If you submit a partial solution, please detail all the things you did not implement: this will help us evaluate your sense observation. You have to ask yourself questions like "is the app state persisted?", or "is the app responsive?"

You will have to use the following technologies (that are available in the CodePen)

  • ReactJS (with JSX syntax, preferable ES6)
  • Underscore
  • LESS

Please try to use these technologies the way they are supposed to be. When using LESS, use nesting and modern CSS (no floats). When using React/Underscore, try to use a functional programming approach the most you can.
